# Games

A collection of AI-generated game experiments made by [lukasvan3l](https://github.com/lukasvan3l) and his kids — Floris and Yune.

All games are built with [Claude](https://claude.ai) as a creative coding partner. The ideas and designs come from us; Claude helps turn them into working code.
